### 2. 创建合法短链接

验证对象：

1. `postShortLinks`
2. `JsonShortLinkRepository`

输入：

```json
{
  "method": "POST",
  "body": {
    "url": "https://example.com/articles/123"
  }
}
```

期望结果：

1. 返回状态码 `201`。
2. 响应体包含 `shortCode`。
3. 响应体包含 `shortUrl`。
4. `shortUrl` 符合 `https://s.example.com/{6位短码}`。
5. 初始 `clickCount` 为 `0`。

实际结果：

1. 返回状态码 `201`。
2. 生成示例短链接：`https://s.example.com/2nPSJA`。
3. `clickCount = 0`。

### 3. 拒绝危险协议 URL

验证对象：

1. `postShortLinks`
2. `validateShortLinkUrl`

输入：

```json
{
  "method": "POST",
  "body": {
    "url": "javascript:alert(1)"
  }
}
```

期望结果：

1. 返回状态码 `400`。
2. 错误码为 `UNSUPPORTED_URL_SCHEME`。
3. 不创建短链接记录。

实际结果：

```json
{
  "code": "UNSUPPORTED_URL_SCHEME",
  "message": "URL scheme is not supported."
}
```

### 4. 不存在 code 返回 404

验证对象：

1. `getShortLinkByCode`
2. `JsonShortLinkRepository`

输入：

```json
{
  "method": "GET",
  "params": {
    "code": "NOPE12"
  }
}
```

期望结果：

1. 返回状态码 `404`。
2. 错误码为 `SHORT_LINK_NOT_FOUND`。
3. 不返回 `Location` header。
4. 不增加访问次数。

实际结果：

```json
{
  "status": 404,
  "body": {
    "code": "SHORT_LINK_NOT_FOUND",
    "message": "Short link was not found."
  }
}
```

```json

  "originalUrl": "https://example.com/expired",

  "clickCount": 0
}
```

当前时间：

```text
2026-01-01T00:00:00.000Z
```

期望结果：

1. 返回状态码 `410`。
2. 错误码为 `SHORT_LINK_EXPIRED`。
3. 不返回 `Location` header。
4. 不增加访问次数。

实际结果：

```json
{
  "status": 410,
  "body": {
    "code": "SHORT_LINK_EXPIRED",
    "message": "Short link has expired."
  }
}
```

### 6. 有效 code 返回 302

验证对象：

1. `getShortLinkByCode`
2. `JsonShortLinkRepository`

前置数据：

通过 `postShortLinks` 创建一个有效短链接，原始链接为：

```text
https://example.com/live
```

输入：

```json
{
  "method": "GET",
  "params": {
    "code": "创建接口返回的 shortCode"
  }
}
```

期望结果：

1. 返回状态码 `302`。
2. `Location` header 等于 `https://example.com/live`。
3. 不返回 JSON 错误体。

实际结果：

```json
{
  "status": 302,
  "headers": {
    "Location": "https://example.com/live"
  }
}
```
